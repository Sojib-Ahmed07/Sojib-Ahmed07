## 🛠️ Production-Ready Case Studies

### 🛒 [NexMart - Multi-Vendor Marketplace](https://github.com/Sojib-Ahmed07/nexmart)
> **A scalable multi-vendor ecosystem with strict, secure backend validation processes.**
*   **Multi-Role Panels:** Engineered isolated experiences and dashboards for Shoppers, Sellers, and Admins.
*   **Secure Payment Architecture:** Native SSLCommerz processing protected with transactional guards to eliminate double-processing.
*   **Session-Backed Security:** All session structures are verified server-side on PostgreSQL (not just cookie dependent), using Better Auth.
*   **Tech Stack:** Next.js 16, TypeScript, PostgreSQL, Prisma ORM, Better Auth, SSLCommerz, Cloudinary, Brevo, Tailwind CSS v4, Vercel.
*   🔗 **[Launch Application](https://nexmart-labs.vercel.app)**

### 🎨 [Craftora - Full-Stack E-Commerce](https://github.com/Sojib-Ahmed07/craftora)
> **An agile, end-to-end e-commerce store backed by visual administration tracking components.**
*   **Visual Analytics:** Features real-time administrative dashboards built with Recharts displaying sales data and order statuses.
*   **Session Scopes:** Secured using strict role-based authentications powered by Better Auth engines.
*   **Tech Stack:** Next.js, TypeScript, Better Auth, Prisma ORM, PostgreSQL, NeonDB, Recharts, Vercel.
*   🔗 **[Launch Application](https://craftora-kohl.vercel.app)**
